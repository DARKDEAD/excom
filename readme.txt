В файле .env следующее сожержание
DEVELOPMENT_DATABASE_URI = "mysql+mysqlconnector://root:__USER__@localhost/excom"
SECRET_KEY = '_________________________'

Запустить из корня проекта: python run.py

Войти в ЛК, пользователь: admin@admin, пароль любой.

Выбрать "Импорт" и указать файл для загрузки.
